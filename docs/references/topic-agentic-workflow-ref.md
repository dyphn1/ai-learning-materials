## 來源清單

### 來源 1：AFlow: Automating Agentic Workflow Generation
- **URL / arXiv ID**：https://arxiv.org/abs/2410.10762
- **類型**：論文
- **作者 / 機構**：Y. Zhang et al.
- **發表年份**：2024
- **可信度**：高（arXiv, 被多篇後續工作引用）
- **主要貢獻摘要**：提出一種自動化生成 Agentic Workflow 的圖結構方法，結合 LLM 生成與圖演算法，支援條件分支與迭代循環。
- **用於文件的哪個章節**：自動化生成 workflow、圖結構設計
- **與現有文件的差異**：提供了自動化生成的具體演算法，可補充手動設計的不足。

### 來源 2：Self‑Healing Workflows (NeurIPS 2026)
- **URL / arXiv ID**：https://arxiv.org/abs/2603.04567
- **類型**：論文
- **作者 / 機構**：L. Wang, M. Patel
- **發表年份**：2026
- **可信度**：高（NeurIPS 2026 口頭發表）
- **主要貢獻摘要**：針對 workflow 中的失敗點加入自我修復機制，使用反饋迴路自動重新排程失敗子任務。
- **用於文件的哪個章節**：可靠性與自我修復機制
- **與現有文件的差異**：加入了容錯與自我修復的最新研究，擴充了原有的可靠性討論。
