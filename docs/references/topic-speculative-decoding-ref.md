# Speculative Decoding 參考資料

## 2025‑2026 重要論文與資源
- **Speculative Decoding and Beyond: An In‑Depth Survey of Techniques** (arXiv:2502.19732) – 系統性分類與評估各種生成‑驗證框架。\
- **Fast Inference from Transformers via Speculative Decoding** (arXiv:2211.17192) – 原始提出方法。\
- **Scaling LLM Speculative Decoding: Non‑Autoregressive Forecasting in Large‑Batch Scenarios** (arXiv:2511.20340) – 引入非自迴歸草稿模型（SpecFormer）。\
- **TurboSpec: Closed‑loop Control for Speculative Decoding** (EECS‑2025‑224) – 動態調整草稿長度與驗證頻率的系統。\
- **Speculative Speculative Decoding (SSD)** (OpenReview) – 草稿同時預測驗證結果，進一步減少序列依賴。\
- **Efficient LLM System with Speculative Decoding** (Berkeley Tech Report 2025) – 大規模部署實驗與 goodput 度量。\
